\# Transformation Tracker

Une application web pour suivre sa transformation physique (poids, mensurations, progression).



\## Fonctionnalités

\- Suivi du poids et affichage sous forme de graphique.

\- Backend en \*\*FastAPI\*\* avec base de données \*\*SQLite\*\*.

\- Frontend en \*\*React\*\* avec Vite.



\## Prérequis

\- Python 3.14+

\- Node.js 20+

\- Git
